# Navigating Ripple

Learn how to move around the app and access all features.

## Main Navigation

### Desktop (Sidebar)
- **Dashboard**: Overview of your activity and stats
- **Contacts**: Manage your network
- **Discover**: Find new connections
- **Meetings**: Schedule and track meetings
- **Goals**: Set and track networking goals
- **Reminders**: View upcoming follow-ups
- **Profile**: Manage your profile and settings

### Mobile (Top Menu)
- Tap the menu icon to access all pages
- Same pages as desktop, optimized for mobile

## Page Structure

Most pages follow this structure:
- **Header**: Page title and main action button
- **Content Area**: Main content (lists, cards, forms)
- **Sidebar/Detail Panel**: Additional information (on some pages)

## Quick Actions

- **Add Contact**: Available from Contacts page
- **Add Meeting**: Available from Meetings page
- **Create Goal**: Available from Goals page
- **Search**: Available on Contacts and Discover pages

## Keyboard Shortcuts

- Use Tab to navigate between form fields
- Press Enter to submit forms
- Use Escape to close dialogs

## Tips

- Use the sidebar to quickly switch between pages
- The Dashboard gives you a quick overview
- Most pages have search and filter options
- Use the back button in your browser to go back
- Click on contact names to see full details

## Getting Help

- Use the chatbot assistant (bottom right) for navigation help
- Hover over help icons (?) for field explanations
- Check example walkthroughs when available
- Review tooltips for quick tips
